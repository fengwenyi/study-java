## java 8 Optional

#### Optional.empty()

创建空Optional

#### Optional.of()

创建一个不是空的Optional

#### Optional.ofNullable()

创建一个允许为空的Optional

#### isPresent()

判断是否有值

#### get()

获取值

#### map()

取值

#### orElse()

为空时，设置默认值

#### orElseGet()

为空时，设置默认值（值为空时，才调用默认值的方法）

#### orElseThrow()

如果值为空则抛出异常

#### filter()

过滤
